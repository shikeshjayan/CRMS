# Customer Relationship Management (CRM)

A full-stack **Customer Relationship Management (CRM)** System application built with modern web technologies.  
This system helps manage customer information, interactions, and business relationships efficiently.

---

## 🚀 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB

### Tools
- Git & GitHub
- REST APIs
- Environment Variables

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/shikeshjayan/CRMS.git

cd CRMS

cd server
npm install

cd ../client
npm install

PORT={PORT}
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

VITE_API_URL=http://localhost:{PORT}

cd server
npm run dev

cd client
npm run dev
